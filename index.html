FAQ
Frequently asked questions
Do I need an AI API key?
Only for AI features (summaries, risk, drafts). You can use Groq's free tier or any OpenAI‑compatible provider.
Is my data safe?
Yes – everything stays in your browser. No servers, no tracking. Your GitHub and AI keys are stored locally.
Can I use the same license on multiple devices?
Yes – one license per OS user profile. You can deactivate and reactivate up to 3 times per year.
What's the refund policy?
30‑day full refund, no questions asked. If it doesn't save you time, we'll return your money.
Does it work with private GitHub repos?
Yes – PR Focus works with private repositories. Make sure your GitHub token has the repo scope when connecting. Your private code never leaves your browser.
